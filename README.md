# YouTube 多視窗播放器

一個讓你同時觀看多個 YouTube 影片的網頁應用程式。完美支援影片比較、多直播觀看，或是打造你專屬的多視窗觀看體驗。

## 功能特色

- 🎥 同時觀看多個 YouTube 影片
- 📱 根據影片數量自動調整的響應式網格佈局
- 🔊 預設自動播放但靜音（符合瀏覽器政策）
- ▶️ 所有影片同步播放控制
- 📋 一鍵複製分享連結
